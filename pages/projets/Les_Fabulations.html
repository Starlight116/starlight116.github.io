Conçu avec :

  • Les étudiants de M2 DIMI
WebDesign Programmation Illustration

Ce projet part d'un hackathon où nous avons, pendant les pauses, dessiné sur des post it et collés ceux ci sur un pan de mur dans la salle de travail. Avec le temps, ces dessins se sont transformés en une véritable oeuvre collective qui évoluait au fil des jours.

L'idée était de capturer cette dynamique créative et de la transposer dans un format numérique accessible à tous.

Nous avons donc conçu les "Fabulations", qui reprend ce mur pour exposer les dessins de nos camardes de manière numérique.

Sur ce projet j'ai principalement travaillé sur le code du site web a partir du design partagé par Jules Ligeret.

Le site est réalisé en HTML, CSS et JavaScript pour assurer une expérience fluide. Le design met en avant les dessins en utilisant une mise en page simple et épurée, permettant aux utilisateurs de se concentrer sur le contenu visuel.

Le scroll inversé permet de remonter ce mur de dessins, recréant ainsi l'expérience de découverte progressive que nous avions lors du hackathon. Les post-its sont dans un ordre choisi, tels qu'ils étaient collés sur le mur, et la légère rotation appliquée à chaque dessin ajoute une touche d'authenticité, rappelant l'aspect spontané de l'original.
